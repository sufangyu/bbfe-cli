# bbfe-cli

项目初始化脚手架


## 如何使用

``` bash
# install with npm
npm install -g bbfe-cli

# install with yarn
yarn global add bbfe-cli

# create project
bbfe create <name>
```

## 目录结构

```bash
.
├─bin                   # 执行命令
├─lib                   # 辅助库
├─templates             # 项目模板配置
├─utils                 # 工具类
└─package.json
```
